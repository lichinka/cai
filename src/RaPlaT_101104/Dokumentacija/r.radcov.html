NAME

r.radcov - Radio coverage simulation

KEYWORDS

SYNOPSIS

r.radcov
r.radcov help
r.radcov [-rpc] csv_file=string antmap_file=string dem_map=string [default_height=integer] [clutter_map=string] [default_clutter=integer] [region=string] frequency=integer cellnum=integer out_map=string db_driver=string database=string out_table=string [procnum=integer] [--overwrite]

Flags:

-r
Recalculate all sector radio coverage files
-p
(purge) Delete all unused sector radio coverage files
-c
(check) Test run without actually performing radio coverage computation
--overwrite
Force overwrite of output files

Parameters:

csv_file=string
Radio cell/sector table in CSV format
Default: sector_table.csv
antmap_file=string
Antennas map file
Default: $GISBASE/etc/radio_coverage/antennamap
dem_map=string
DEM file for radio coverage simulation
Default: dem_map@PERMANENT
default_height=integer
Default DEM height
clutter_map=string
Clutter map file (required for Ericsson model)
Default: clutter_map@PERMANENT
default_clutter=integer
Default clutter attenuation
region=string
Computation region (dem,current or region,rast,n,e,s,w,res (see g.region))
Default: dem
frequency=integer
Radio frequency
Default: 900
cellnum=integer
Number of succesive path loss values to be written in the table
Default: 5
out_map=string
Simulated radio coverage - raster (output)
Default: out_raster
db_driver=string
Database driver
Options: none,dbf,mysql,pg
Default: none
database=string
Database name
Default: $GISDBASE/$LOCATION_NAME/$MAPSET/dbf
out_table=string
Simulated radio coverage - db table (output)
Default: out_db
procnum=integer
Number of parallel processes (-1: automatic, 0: non-parallel)
Default: -1
